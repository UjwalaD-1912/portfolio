### 1. Personal Information
Update the following in `index.html`:

```html
<!-- Update name in navigation -->

<!-- Update hero section -->

<!-- Update contact information -->
<span class="method-value">your.email@example.com</span>
<span class="method-value">linkedin.com/in/yourprofile</span>
```

### 2. Skills and Experience
Edit the skills section in `index.html`:

```html
<!-- Update skill percentages -->
<div class="progress" style="width: YOUR_PERCENTAGE%"></div>

<!-- Add/remove skill categories -->
<div class="skill-category">
    <h3 class="category-title">Your Category</h3>
    <!-- Add skill bars -->
</div>
```

### 3. Projects
Customize the projects section:

```html
<div class="project-card">
    <div class="project-image">
        <i class="fas fa-your-icon"></i> <!-- Change icon -->
    </div>
    <div class="project-content">
        <h3>Your Project Name</h3>
        <p>Your project description...</p>
        <div class="project-tech">
            <span class="tech-tag">Technology 1</span>
            <span class="tech-tag">Technology 2</span>
        </div>
    </div>
</div>
```

### 4. Certifications
Add your Salesforce certifications:

```html
<div class="certification-card">
    <div class="cert-badge">
        <i class="fab fa-salesforce"></i>
    </div>
    <h3>Certification Name</h3>
    <p>Certification description...</p>
    <span class="cert-date">Earned: YEAR</span>
</div>
```

### 5. Color Customization
Modify CSS variables in `styles.css`:

```css
:root {
    --sf-blue: #YOUR_PRIMARY_COLOR;
    --sf-orange: #YOUR_ACCENT_COLOR;
    /* Update other colors as needed */
}
```
